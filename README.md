# projeto-hyrox-planner

As instruções para o projeto podem ser encontradas [AQUI](https://drive.google.com/file/d/1qVGanEIEH4xpZSDynaT9wVjXr9r0z52t/view?usp=sharing)

Projeto desenvolvido por Ana Júlia Melo, Ana Luiza Câmara, Arthur Marenga, Beatriz Vieira, Laís Moura e Victor Telles.

O HYROX é uma competição global de corrida fitness (conhecida como World Series of Fitness Racing) que combina corrida e exercícios funcionais. O formato padrão consiste em 8 km de corrida total, intercalados com 8 estações de exercícios de força (uma a cada 1 km).

Este projeto tem como objetivo auxiliar no acompanhamento e organização dos treinos para as competições da modalidade. Ele engloba o registro e a evolução dos treinos e exercícios cadastrados pelo usuário, além de permitir o rastreamento da evolução dos simulados HYROX.
